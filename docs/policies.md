# Policies v0.4

## Emergency Fund

Target: €20,000.

## Cash Allocation Engine

1. Planned expenses are reserved first.
2. If Emergency Fund is below target:
   - <50% funded: 35% of available income to Emergency
   - 50–80% funded: 25%
   - 80–100% funded: 15%
3. Investments receive minimum 12% of available income.
4. User may edit a scenario before posting.
5. Post is blocked if proposal total does not equal income.
